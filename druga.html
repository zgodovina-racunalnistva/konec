DRUGA GENERACIJA RAČUNALNIKOV (1953 - 1974)

Izum tega obdobja je bil tranzistor.

Slika 13: Tranzistor

Tranzistor je bil v primerjavi z elektronkami:

  • hitrejši,
  • manjši,
  • zanesljivejši,
  • energetsko učinkovitejši in
  • cenejši.

Leta 1947 je William Shortthy v Združenih državah Amerike izdelal prvi tranzistor (zato katerega je prejel Nobelovo nagrado), ki je predstavljal osnovni gradnik druge generacije računalnikov. Tranzistor je predstavljal revolucijo zaradi svoje majhnosti, zanesljivosti in majhne porabe el. moči. Cena računalnikov je močno padla, postali so manjši, manj občutljivi. Izdelali so že prve prenosne računalnike.

Vsak nov izum ima tudi določene posledice - to velja tudi za tranzistor. Nekatere med njimi so naštete spodaj

  • računalniki so postajali vedno manjši,
  • bolj zanesljivi in zmogljivejši,
  • pričeli so uporabljati zbirne jezike,
  • pojavili so se magnetni diski in
  • ljudje so začeli razvijati višje strojne jezike.

Prvi tranzistorski splošno - namenski računalnik je bil TRADIC.
